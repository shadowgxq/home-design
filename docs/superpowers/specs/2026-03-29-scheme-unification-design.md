# 方案统一化设计文档

日期：2026-03-29

## 背景

当前项目中的空间、门窗、卫生间、色彩、吊顶和示意图文档经历过多轮讨论，已经出现跨文档口径不一致的问题。最明显的冲突集中在“电竞房是否在中庭内二次隔断”这一点上。

用户已明确确认以下事实：

* 中部空间为**纯开放多功能中庭**。
* 北侧独立房间**直接就是电竞房**。
* 当前方案中**不存在**“中庭里再隔一个电竞房”的逻辑。
* 电竞房采用**普通房门逻辑**，不再使用玻璃门、吊轨门、展示柜一体方案。
* 客厅南门以 `2800 × 2400`、`两轨两扇重型推拉` 为准。
* 客厅北门以 `2200 × 2400`、`三联动推拉` 为准。
* 卫生间方案继续保留“南沐浴 / 北如厕家政”。

本设计文档的目标，是在不引入新方案分支的前提下，建立一个唯一有效的当前口径，并定义后续文档清理规则。

## 统一口径

### 空间定义

* 客厅：南向主起居空间。
* 中部空间：开放式多功能中庭，不做玻璃盒子、不做电竞房过道隔断。
* 北侧独立房间：电竞房，本身是一个完整房间。
* 北侧西房间：杂物间 / 收纳中心。

### 门洞定义

* 客厅南门：`2800 × 2400`
  * 开启方式：两轨两扇重型推拉
  * 调整逻辑：西扩300 + 东扩900
* 客厅北门：`2200 × 2400`
  * 开启方式：三联动推拉
  * 调整逻辑：西不动 + 东扩600
* 电竞房门：普通房门逻辑
  * 不再定义为玻璃门、吊轨门、展示柜一体门
  * 若未单独指定尺寸，则沿“普通房门体系，优先统一高度”的口径处理

### 卫生间定义

* 南卫：纯沐浴与护理区，`960 + 960` 指左右功能分区，不是 `960 × 960` 成品淋浴房盒子。
* 北卫：如厕 + 洗衣 + 简洗功能复合空间。
* 卫生间门目前不能再视为“已完全定案”，后续整理中应避免把“内推”写成唯一正确答案。

## 文件处理原则

### 直接删除

凡是建立在“中庭里再隔一个电竞房”这一前提上的内容，全部删除，不保留正文，不保留并行方案，不保留“推荐但暂不采用”的模糊措辞。

适用内容包括：

* 电竞房玻璃门
* 吊轨移门
* 两侧砌墙 + 中间玻璃门
* 玻璃盒子
* 展示柜一体门
* 中庭内二次隔断电竞房
* 为该隔断前提服务的材质、尺寸、造价、关键词建议

### 改写保留

凡是仍能服务当前方案的审美逻辑、材料逻辑、尺度逻辑，可以改写后保留，但必须去除对失效隔断方案的依赖。

适用内容包括：

* 黑色金属线条系统
* 公共区高门洞统一语言
* 玻璃材质的通透性原则
* 与客厅南北门相关的尺寸、比例和开启方式建议
* 卫生间门的材质与隐私逻辑

### 原样保留

凡是已经符合当前唯一口径的内容，不做风格化重写，只做必要交叉引用修正。
