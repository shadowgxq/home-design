# 方案统一化设计文档

日期：2026-03-29

## 背景

本设计文档只保留当前唯一有效方案，用于统一项目中的户型、布局、色彩、门窗、吊顶与灯光口径。

## 当前唯一口径

### 基础坐标与尺寸

* 图纸上方 = 南
* 图纸下方 = 北
* 图纸左侧 = 东
* 图纸右侧 = 西
* 客厅基础尺寸：`47.20m² / 7750 × 6090`
* 主卧基础尺寸：`26.70m² / 3570 × 7480`

### 空间关系

* 客厅位于东侧，主卧位于西侧。
* 客厅西墙为悬浮电视背景墙。
* 客厅东墙为满墙高柜。
* 中部空间为开放式多功能中庭。
* 北侧独立房间为电竞房。
* 北侧东房间为超级收纳中心。

### 客厅口径

* 客厅西墙：悬浮电视背景墙，沙发正向面向西墙。
* 客厅东墙：满墙高柜。
  * 中部设置留空壁龛作为茶咖操作区。
  * 中部留空壁龛与电器操作台采用深炭灰木纹一体处理。
  * 柜门大面采用暖白 / 羊脂白。
  * 东墙原窗封掉。
  * 东墙南端预留立式空调机位。
  * 茶咖操作区不做上下水。
  * 茶咖操作区保留灯光与操作插座。

### 主卧口径

* 主卧门采用深炭灰木纹木门。
* 主卧东墙为整排衣柜墙。
* 主卧西墙为睡眠背景墙。
* 主卧入口改为纯封闭收纳。
* 主卧整排柜体以封闭柜门为主，仅梳妆位做深色跳出。
* 主卧采用简单单层平顶下吊。
* 主卧南向只有窗，没有门。
* 主卧与阳台区直接打通，按一个完整房间处理。
* 主卧窗帘做到最外侧。
* 主卧与阳台地面同材连续。
* 主卧西南角设置立式空调。

### 门洞与门型

* 客厅南门：`2800 × 2400`，两轨两扇重型推拉。
* 客厅北门：`2200 × 2400`，三联动推拉。
* 电竞房门：普通房门逻辑，深炭灰木纹木门。
* 卫生间门：长虹玻璃 + 黑色极窄边框，两扇均向内开启。

### 吊顶与灯光

* 客厅与中庭：悬浮大平顶。
* 主卧：简单单层平顶下吊。
* 主照明统一 `4000K`。
* 氛围灯与地脚灯允许 `3000K`。
* 客厅中央保留 `4盏高功率宽光束射灯` 作为基础补光。

### 色彩系统

* 墙面与顶面：菱花白。
* 柜门大面：暖白 / 羊脂白。
* 木门、电视柜、茶咖操作区核心、局部跳色：深炭灰木纹。
* 金属线条：黑色或深灰。

## 文档执行原则

* 旧口径不保留对照说明，直接删除。
* 所有正式文档只描述当前方案，不描述历史分支。
* `svg` 与示意图不在本轮处理范围内。

## 验收标准

完成后应满足以下条件：

* 正式方案文档中不再出现彼此冲突的尺寸、方位、墙面功能与门型口径。
* 客厅西墙电视主墙、客厅东墙满墙高柜、主卧打通阳台三条主逻辑可以同时成立。
* 主卧门、电竞房门、客厅东墙茶咖操作区、电视柜的深色材质系统可以形成统一闭环。
