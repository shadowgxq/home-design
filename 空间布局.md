# 空间布局规划

## A.1 客厅区域 (Living Room)
*   **面积**：48.8m² (8.0m × 6.1m)
*   **入户特征**：北侧入户门居中（中轴线）

### 1. 视听核心区 (Media Zone) - 西侧
*   **电视悬浮背景墙 (East Wall)**
    *   **位置与尺寸**：利用主卧衣柜背面（东墙），打造超级背景墙
    *   **厚度规划 (Critical Dimension)**：**总厚度预留 650mm - 700mm**
        *   **计算公式**：衣柜净深(550mm) + 柜门(20mm) + 背板结构层(50mm) + 饰面层(15mm) ≈ 635mm。
        *   **余量**：建议直接放线 **650mm** 或 **700mm**，多余空间用于填充隔音棉及修正施工误差，确保体量感。
    *   **设备推荐**：85寸电视机、悬浮背景板、落地电视柜
*   **沙发位 (Seating)**
    *   **摆放**：居中摆放，面向东墙
    *   **选品**：推荐低靠背意式模块沙发，总宽3.2m-3.6m
    *   **间距**：沙发靠背距离电视墙约3.5-4m

#### 电视悬浮背景墙详细尺寸
*   **整体宽度设计**：
    *   85寸电视区域核心尺寸：`1940mm * 1130mm`
    *   左右展示柜子：宽度 `650mm`
    *   白色墙体部分：`880mm`
    *   居中设计：左右各 `500mm`
*   **高度设计**：
    *   地台：`200mm` (保持不变)
    *   悬浮缝隙：`150mm` (保持不变)
    *   板体总高：`2050mm` (2.05米)
    *   下白边：`320mm` (微增，更稳重)
    *   中黑区：`1130mm` (电视不变)
    *   上白边：`600mm` (增加高度，填补空缺)
    *   顶部留空：`600mm`
*   **电视机内嵌尺寸**：
    *   选择固定支架（首选）：`120mm - 130mm`
    *   选择伸缩支架：`150mm - 160mm`
    *   分层建议：70内侧+50外侧 或 90内侧+60外侧
*   **视距参考**：
    *   沙发靠背距离电视墙：3.6-4米
    *   净视距：3.5米（减去沙发厚度）

### 2. 东墙：极简全墙收纳与茶饮中心 (The East Wall System) - 6m
*   **核心决策**：**全墙柜体 + 隐形封窗**。
    *   *理由*：原有窗户破坏极简视觉完整性且易眩光。必须通过轻钢龙骨+石膏板（或砌墙）做内墙遮蔽，打造纯粹、完整的背景墙面。
*   **材质配色**：
    *   **主调 (80%)**：肤感暖白 PET (与墙面同色)，一门到顶，无拉手设计（反弹器/45°内切）。
    *   **跳色 (20%)**：烟熏橡木 / 深摩卡色 / 哑光黑 (用于茶吧壁龛或展示区)，构建视觉重心。
*   **功能分区布局**：
    *   **A. 储物与展示 (两侧)**：封闭式高柜负责囤货收纳；可选黑色铝框灰玻门展示酒具，内嵌4000K线性灯。
    *   **B. 核心茶吧台 (中部)**：留空壁龛 (宽约2.0m-2.4m，高60-80cm)。
        *   *配置*：岩板台面，预留滑轨插座，作为净饮机、咖啡机操作区。
        *   *互动*：中心对齐沙发后方的大板茶桌，形成便捷的“取水-泡茶”动线。
    *   **C. 空调位 (最右侧 1.0m -> 建议优化为 800mm)**：
        *   **尺寸界定**：
            *   **舒适红线 (800mm)**：这是平衡收纳与美学的最佳尺寸。
                *   *计算逻辑*：圆柱柜机直径约 400-450mm + 窗帘位 200mm + 散热/维修/视觉呼吸缝隙 150mm。
            *   **极限底线 (650mm)**：仅能塞入机器，维修困难，回风噪大，视觉局促（**强烈不推荐**）。
            *   **宽裕型 (1000mm)**：若储物需求不急迫，留 1m 可在空调旁摆放大型绿植（量天尺），打造“虽然我有柜子，但我很透气”的松弛感。
        *   **软装**：在柜体侧板与空调之间，利用余量布置绿植遮挡管线。
*   **施工要点**：
    *   **封窗处理**：窗户玻璃务必贴防爆膜，再做假墙遮蔽。

### 3. 沙发背几
*   **合理性深度分析 (Rationality Analysis)**：
    *   **结论**：**极度合理，尤其适合您的大横厅**。
    *   **Why Yes (优势)**：
        *   **视觉压舱石**：解决沙发背后空荡荡的不安全感，增加空间的层次感和厚重感。
        *   **交流零阻碍**：工作/喝茶时抬头就能看到电视和家人，避免了传统书房的“面壁思过”和“家庭隔离感”。
        *   **洄游动线**：强制形成“回”字形动线，让气流和人在空间中流动更顺畅。
    *   **Why No (潜在风险与解决方案)**：
        *   *风险*：电源难接。 *解法*：装修阶段预埋**地插**，或沿沙发底部走线至桌腿。
        *   *风险*：显得凌乱。 *解法*：桌子深度控制在 60cm 以内，且必须配备**强收纳功能的各种盒子**（收纳遥控器/数据线）。
*   **布局设计实操 (Layout & Specs)**：
    *   **尺寸黄金法则**：
        *   **长度**：`2400mm` (假设沙发3.2m，两边各留40cm呼吸空间)。
        *   **深度**：`500mm - 550mm` (核心！太宽像餐桌，太窄放不下笔记本。55cm是兼顾书写与美感的极限)。
        *   **高度**：`750mm` (标准书桌高，沙发靠背通常在650-800mm，桌子略低于或齐平于沙发靠背视觉最舒适)。
    *   **选材建议**：
        *   **桌面**：5cm厚 **黑胡桃木直拼板** (大板)，保留木纹质感，边缘做斜切工艺 (显得轻盈)。
        *   **桌腿**：亚克力悬浮腿 (极简) 或 黑色实木片腿 (稳重)。
    *   **配套软装**：
        *   **灯光**：桌面上必须有一盏**有雕塑感的台灯** (如 Flos Snoopy 或 陶瓷底座灯)，作为中景照明。
        *   **座椅**：2把 **昌迪加尔椅 (黑色藤编)** 或 **中古悬臂椅**。平时必须能完全推入桌底，不占过道。
    *   **电力系统 (The Universal Power Strategy)**：
        *   **关于“明线+隐藏条”的避坑警告**：坚决**不建议**从墙面拉明线并用隐藏条/压线槽贴在地面。大横厅的核心优势是地面的无界延伸感，隐藏条不仅会像一道疤痕割裂空间，更会在高频的“洄游动线”上制造绊脚隐患，且成为扫地机器人的清洁死角。
        *   **设计原则**：**沙发决定权归您，地插只做配角**。绝不能因为一个插座限制了您选择“云朵沙发”或“豆腐块沙发”的自由。
        *   **进阶落位方案：抛弃传统地插，采用“极简出线孔”或“下沉式地插”**
            *   **方案A：下沉式隐藏地插（灵活）**。插头插在凹槽内部，盖板合上后地面依然平整，不怕扫地机碰撞。
            *   **方案B：极简出线孔直连（极致隐形，性价比最高）**。地面不装插座面板。预埋一根50mm的PVC管直通墙面，一根超长的黑色插排线直接从地面穿出，紧贴着桌腿固定。0机械故障率，视觉最干净。
            *   **最佳坐标（针对方案A/B）**：**书桌右侧桌腿内侧 20cm，且位于桌板深度中心**。
            *   **为什么这是最优解？**
                1.  **全兼容**：无论您买高脚沙发还是落地沙发，这个位置都在桌子下面，互不干扰。
                2.  **防踢保护**：桌腿是天然的挡箭牌。人的脚不会去踢桌腿，形成了“死角保护区”。
                3.  **走线极短**：线出来直接顺着桌腿爬上去，视觉上几乎隐形。
        *   **【关键】坐标落地算法 (The Coordinate Calculation)**：
            *   **痛点**：位置一旦算错，地插就会被压在沙发腿下，或露在外面很难看。
            *   **公式**：`地插离电视墙距离` = `最佳观影视距` + `沙发靠背厚度` + `(桌板深度 ÷ 2)`
            *   **实操举例**：
                *   假设电视墙完成面到沙发靠背视距为 **3500mm** (大横厅黄金视距)
                *   假设您看中的沙发靠背厚度为 **300mm**
                *   假设书桌深度为 **550mm** (一半即275mm)
                *   **地插中心点位置** = 3500 + 300 + 275 = **离电视墙 4075mm 处**。
            *   **容错技巧 (The Tolerance Trick)**：
                *   **宁深勿浅**：如果不确定沙发具体款式，就把地插位置**往桌子深处多算 10cm**。桌子盖住地插是最好的，露出来是最丑的。
                *   **左右定位**：不要放在正中心！一定要放在**侧边桌腿内侧 20cm**。正中心容易和人腿打架，侧边才是死角。

## A.2 主卧区域 (Master Bedroom)
*   **面积**：24.87m² (3.32m × 7.49m)

### 布局细节 (Master Plan for Master Bedroom)

1.  **东墙：7m 极简收纳与梳妆系统 (East Wall System)**
    *   **A区 (入口 1.0m)**：**次净衣/展示区** (极窄黑框灰玻门)。
        *   *尺寸落地*：设计为**两扇对开门，单扇门宽约 500mm**，在3米层高下确保受力安全及修长的视觉比例。
    *   **B区 (睡眠区 3.0m)**：**主力衣柜** (一门到顶 PET 肤感白)。
    *   **C区 (中段 1.8m)**：**壁龛式梳妆/阅读台** (原墙体位置)。
        *   *设计*：内嵌台面 + 智能灯带，作为“呼吸口”打破长柜沉闷。
    *   **E区 (阳台区 1.15m)**：**家政/囤货柜** (全封闭)。

2.  **西墙：睡眠岛与L型分区背景 (The Wood Box & The White Box)**
    *   **总尺寸**：2.05m入口过渡 + 3.8m睡眠岛 + 1.15m纯白留白 = 7.00m
    *   **方案核心**：**3.8m 木质睡眠岛 + L型折边 + 1.15m 极致留白**
        *   **入口过渡区 (2.05m)**：**极简白墙与前后层次 (The White Entry)**。
            *   **材质**：**纯白哑光乳胶漆**（#F5F5F5 / NCS S 0502-Y），**不做木工基层**。
            *   **尺寸逻辑**：2.05m 恰好是能够容纳一个 1.2m 条案 + 两侧呼吸空间的最小舒适宽度。
            *   **机电**：条案位预留 **双五孔 + Type-C** 插座；顶嵌筒灯 + 极窄壁洗光 (3000K)。
        *   **睡眠区 (3.8m)**：**全高背景墙 (The Wood Box)**。
            *   **材质**：下半墙深灰硬包 + 上半墙暖灰木纹 + 中间灯带。
            *   **图1美学植入 (关键优化)**：在北侧 (靠入口处) 增加 **600mm 竖向极简木格栅**。
                *   *形式*：**全高落地格栅 (Floor-to-Ceiling)**。
                *   *细节*：这 600mm 区域不做上下分色，而是从地面一直通顶到天花板，作为“木盒子”的强势开端。
            *   **范围**：从入口一直延伸到原阳台墙体位置（约3.8m处）。
        *   **L型折边 (关键细节 - 阳台侧)**：
            *   **位置**：睡眠区与阳台白墙的交界处（约 5.85m 处）。
            *   **形态**：**睡眠区背景墙**（中间 3.8m）整体向前凸出 **100mm**，形成侧面收口。
            *   **视觉效果**：就像一块巨大的厚木板（睡眠背景）“悬浮”在白墙之上。
            *   **工艺**：45°海棠角拼接，与白墙留3mm自然缝。
        *   **阳台区 (1.15m)**：**极致紧凑留白 (Pure White)**。
            *   **布局**：西南角放立式空调 (0.5m) + 紧邻斜放 Eames Lounge Chair。
            *   **效果**：虽然紧凑，但白墙背景让空间显大，松弛感拉满。

3.  **阳台与空调 (Balcony & AC)**
    *   **结构**：通过 **2800×2400 两轨两扇重型推拉门** 连接阳台，保持南向通透，但**不做完全无门洞的一体化打通**。
    *   **空调**：立式空调置于 **西南角 (阳台西墙角落)**，背靠白墙。
    *   **窗帘**：安装在 **最南端阳台窗户** (梦幻帘/电动纱帘)。

4.  **北侧入户：功能化过渡区**
    *   **定位**：**次净衣管理区** 或 **极简玄关景**。
    *   **材质**：**纯白哑光乳胶漆**，**不做下半墙硬包与上半墙木饰面**，入口保持轻盈与通透。
    *   **防污与耐用**：建议在 **900–1000mm 高度**做窄腰线（同色肤感板或微水泥薄涂）。
    *   **基层找平 (关键工艺)**：**不做木工基层**，采用 **油工冲筋找平**，确保白墙平整如镜。
    *   **家具选型**：**极窄条案**（1200×300×750mm）或 **落地全身镜**。
    *   **线性边界**：与 **3.8m 睡眠岛折边**形成清晰转折，保证“白区 → 木区”的极简几何逻辑。

## A.3 多功能中庭 (Central Atrium) - 原多功能室
*   **总面积**：38.53 m² (9.24m × 4.17m)
*   **设计决策 (Crucial Decision)**：**取消所有隔断，保留极致大平层体验**。
    *   *背景*：由于北向两个房间（各20m²）已纳入装修范围，电竞房功能可完美移至北向备用房。
    *   *获益*：彻底释放 9.24米 的超大横向面宽，形成 **“南客厅(47m²) + 中庭(38m²)”** 的双巨厅格局。这种空间尺度在豪宅中都极为罕见，任何隔断都是对空间感的浪费。

### 1. 空间功能重构 (Open Layout)
整个中庭不再切割，而是通过家具和功能分区定义空间：

*   **Zone 1: 东侧·端景与交通轴**
    *   **位置**：连接客厅北门、卫生间与北向房间的交通枢纽。
    *   **视觉重心**：
        *   **北墙 (The Focal Wall)**：从客厅看出来的第一眼。悬挂巨幅抽象画 (150cm x 180cm) 或设置端景台。
        *   **灯光**：偏光洗墙灯均匀照亮北墙，营造安静、克制的端景氛围。
*   **Zone 2: 西侧·轻运动与阅读区 (Wellness & Lounge)**
    *   **位置**：原电竞房位置，靠西窗。
    *   **布局建议**：
        *   **跑步机/划船机**：直接面向西窗摆放。运动时视野开阔，不再憋屈在柜子里。
        *   **懒人沙发/单人椅**：配合落地灯，打造一个开放式的阅读角或发呆区。
        *   **极简收纳**：西墙低矮长条柜，用于收纳瑜伽垫、哑铃等小件器材，台面可放置绿植。

### 2. 氛围与材质 (Atmosphere & Materials)
*   **地面**：通铺木地板/地砖，与客厅保持一致，强化延伸感。
*   **墙面**：全屋统一的暖白/浅灰基调，仅在端景墙做材质跳跃。
*   **天花**：**极简大平顶**。采用无主灯设计，两条超长磁吸轨道贯穿东西（9米长），拉伸视觉张力。

### 3. 为什么不隔断？—— 关于“大而空”的辩证思考
您担心的“空间太大、没东西放、显得空”，恰恰是豪宅设计的核心追求。以下是取消隔断的深度理由：

*   **理由一：从“过道”进化为“厅” (From Corridor to Hall)**
    *   **隔断后**：您得到的是一条狭长的“黑过道” + 一个憋屈的“小房间”。过道只是用来走路的，确实没法放东西。
    *   **取消后**：您得到的是一个 38m² 的 **多功能中庭**。它不再是通道，而是一个可以停留、活动、展示的空间。
*   **理由二：采光与通风的绝对正义 (Light & Air)**
    *   **现状**：西侧有窗户。
    *   **隔断后**：西窗的光线被挡在电竞房里，东侧的入户区域（您回家的第一眼）永远是暗的，必须开灯。
    *   **取消后**：西晒的阳光可以一直洒到东侧的卫生间门口。自然光是最好的装修，不花一分钱，但最奢侈。
*   **理由三：健身器材的“尊严” (Gym Dignity)**
    *   **场景**：您有跑步机/划船机。
    *   **隔断后**：健身器材只能塞在小房间的角落，或者面对着墙跑，像在“坐牢”。
    *   **取消后**：健身器材摆在西窗前，面对着窗外的风景跑。旁边放一株 1.8米高的量天尺，放一把休战时的躺椅。这不再是“器材存放处”，而是 **“私人景观健身房”**。
*   **理由四：社交溢出效应 (Social Overflow)**
    *   当家里来朋友聚会（超过6人），客厅坐不下时，人群会自然流向这个宽敞的中庭。有人在客厅看电视，有人在中庭靠着懒人沙发聊天。这种“聚而不乱”的从容感，只有大空间才能提供。

### 4. 软装陈列方案：如何填满空间而不拥挤？
我们用 **“三个视觉岛屿”** 来破解“空”的焦虑，而不是用墙把它堵死：

1.  **东岛 (入户端景)**：
    *   **配置**：**极简玄关条案 (1.5m长)** + **抽象挂画** + **台灯**。
    *   *作用*：一进门（或从客厅出来）看到的不是空墙，而是清晰的端景艺术角。随手放钥匙、香薰。
2.  **中岛 (流动区)**：
    *   **配置**：**留白**。
    *   *作用*：给扫地机器人、给未来的孩子/宠物、给您自己留出一条宽敞的“跑道”。在家里能走直线，是一种极大的心理享受。
3.  **西岛 (活动区)**：
    *   **配置**：**跑步机 (面向窗)** + **单人沙发/躺椅 (面向中庭)** + **落地钓鱼灯** + **大盆绿植 (琴叶榕/龟背竹)**。
    *   *作用*：这组家具本身就是雕塑。它们占据了西侧 15m² 的空间，让空间看起来丰富且有生活气息。

## A.4 北向功能区 (North Zone) - 新增规划

### 1. 独立电竞房 (Gaming Cave) - 原备用房
*   **位置**：北向中间房间 (20.13m²)。
*   **定位**：北侧独立功能房，可作为电竞 / 影音 / 兴趣房使用。
*   **布局要点**：
    *   远离客厅和主卧，适合放置电脑桌、设备与独立空调。
    *   墙面和灯光可比公共区更深、更个性化，但不再延伸为整套中庭隔断方案。

### 2. 超级收纳中心 (Storage Hub) - 原杂物间
*   **位置**：北向西侧房间 (20.13m²)。
*   **定位**：**全屋后勤基地**。
*   **布局建议**：
    *   **U型布局**：三面墙做顶天立地货架/柜体。
        *   **A面**：精密储物 (换季衣物、被褥、行李箱)。
        *   **B面**：家政工具 (吸尘器、梯子、工具箱)。
        *   **C面**：囤货区 (纸巾、矿泉水、粮油)。
    *   **获益**：正因为有了这个20平米的超级仓库，南向的客厅和主卧才能真正做到“空无一物”的极简高级感。

## A.5 专项施工：客卧隔断与悬浮背景墙

### 1. 隔断选型对比
*   **方案A：钢结构+木工板（Steel Frame + Plywood）**
    *   **构造**：40×40镀锌方管焊接井字架 + 高密度岩棉 + 双面封欧松板
    *   **优势**：比砖墙薄（8-10cm），省空间
    *   **劣势**：造价高、隔音略逊，需精确预埋电视挂点

### 2. 85寸电视悬浮墙施工工艺
*   **结构安全**：
    *   **双面背对背结构 (Back-to-Back System)**：
        *   **A面 (主卧侧)**：标准衣柜结构，需升级为 **18mm厚背板** 以增强整体刚性。
        *   **B面 (客厅侧)**：在衣柜背板上打 **30mm-50mm 木龙骨/欧松板基层**。
            *   *作用*：走电线管、安装插座底盒、找平、制造悬浮所需的底部内收空间。
    *   **侧面收口 (The Edge)**：衣柜侧板与背景墙侧边必须使用 **同材质见光板** 或 **金属收口条** 统一包覆，形成完整的 650-700mm 建筑体块，杜绝拼接感。
    *   **隔音处理**：在衣柜背板与客厅背景墙基层之间的空隙填充 **高密度隔音棉**。
    *   砖墙方案：轻钢龙骨垫出造型层，挂架穿过造型层打入砖墙
    *   钢架方案：挂架必须锁在内部方管上，严禁只锁在木板上
*   **悬浮做法**：
    *   底部悬空：背景板离地20cm
    *   内凹踢脚：墙根向内收5cm做黑色不锈钢踢脚
    *   灯光槽：底部预留2cm宽×1cm深铝槽，嵌入3000K COB灯带，柔光罩见光不见灯
