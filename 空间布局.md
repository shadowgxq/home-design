# 空间布局规划

## A.1 客厅区域 (Living Room)
*   **面积**：47.20m² (7.75m × 6.09m)
*   **入户特征**：北侧入户门居中（中轴线）

### 1. 视听核心区 (Media Zone) - 西侧
*   **电视悬浮背景墙 (West Wall)**
    *   **位置与尺寸**：利用主卧衣柜背面（西墙），打造客厅主视线的悬浮电视背景墙
    *   **厚度规划 (Critical Dimension)**：**总厚度预留 650mm - 700mm**
        *   **计算公式**：衣柜净深(550mm) + 柜门(20mm) + 背板结构层(50mm) + 饰面层(15mm) ≈ 635mm。
        *   **余量**：建议直接放线 **650mm** 或 **700mm**，多余空间用于填充隔音棉及修正施工误差，确保体量感。
    *   **设备推荐**：85寸电视机、悬浮背景板、落地电视柜
*   **沙发位 (Seating)**
    *   **摆放**：居中摆放，面向 **西墙电视主墙**
    *   **选品**：推荐低靠背意式模块沙发，总宽3.2m-3.6m
    *   **间距**：沙发靠背距离电视墙约3.5-4m

#### 电视悬浮背景墙详细尺寸
*   **整体宽度设计**：
    *   85寸电视区域核心尺寸：`1940mm * 1130mm`
    *   左右辅助边界：宽度各 `650mm`，只作为电视主墙的侧向收边与留白缓冲，**不默认理解为玻璃展示柜**
    *   白色墙体部分：`880mm`
    *   居中设计：左右各 `500mm`
*   **高度设计**：
    *   地台：`200mm` (保持不变)
    *   悬浮缝隙：`150mm` (保持不变)
    *   板体总高：`2050mm` (2.05米)
    *   下白边：`320mm` (微增，更稳重)
    *   中黑区：`1130mm` (电视不变)
    *   上白边：`600mm` (增加高度，填补空缺)
    *   顶部留空：`600mm`
*   **电视机内嵌尺寸**：
    *   选择固定支架（首选）：`120mm - 130mm`
    *   选择伸缩支架：`150mm - 160mm`
    *   分层建议：70内侧+50外侧 或 90内侧+60外侧
*   **视距参考**：
    *   **推荐值**：沙发靠背距离电视墙完成面按 **3700mm** 控制。
    *   **合理区间**：`3600-3800mm`
    *   **净视距**：约 `3300-3500mm`（减去沙发靠背厚度与人体坐姿前移），适配 85 寸主电视。

### 2. 东墙：满墙高柜与茶咖操作区 (The East Wall Cabinet System) - 6m
*   **核心决策**：**柜体优先 + 封窗/包窗处理 + 中部茶咖功能核 + 南端 1000mm 空调位**。
    *   *理由*：这面墙的任务是承担客厅后勤收纳与小家电操作，不是做第二个主背景。东窗按柜体优先处理，但完成后必须像完整界面，不能做成“封了一块洞”的补丁墙。
*   **尺寸定案**：按 **A区 2.6m + B区 2.4m + C区 1.0m = 6.0m** 执行；若现场总长有误差，**只允许 A区吸收误差**，不要动茶咖核心和空调位。
*   **材质配色**：
    *   **主调 (80%)**：肤感暖白 PET (与墙面同色)，一门到顶，统一平板门。
    *   **深色核心 (20%)**：**深炭灰木纹**，只用于中部留空壁龛背板与电器操作台，形成稳定的功能核心。
*   **功能分区布局**：
    *   **A区 北段封闭收纳 (2.6m)**：暖白高柜负责囤货、杂物、器皿与低频家电收纳。
        *   **门板拆分**：建议做 **5 扇等宽高门**，单扇约 **520mm**，避免单门过宽导致高门变形和立面发笨。
        *   **内部逻辑**：按 **高瓶 / 囤货层板 + 吸尘器 / 工具高柜 + 器皿 / 备用电器** 去分，不做任何开放格。
    *   **B区 中部茶咖功能核 (2.4m)**：作为客厅东墙唯一深色核心。
        *   **盒子总宽**：按 **2400mm** 控制，内部净宽建议 **2200mm**。
        *   **下柜与台面**：操作台高度 **900mm**，台面深度 **600mm**，深炭灰木纹台面承担咖啡机、烧水壶和托盘操作。
        *   **留空壁龛**：净高建议 **700mm**，深炭灰木纹背板 + 内置线性灯，只服务小家电操作，不再叠加展示柜语言。
        *   **上柜**：延续暖白封闭柜门，深度控制在 **350mm** 左右，让深色只留在功能核内部。
        *   **用途**：放茶具、咖啡机、烧水壶等小家电，不做上下水。
        *   **电源**：预留 **轨道插座 600-800mm** 或 **5孔插座 × 3**，集中服务小家电。
    *   **C区 南端立式空调位 (1.0m)**：
        *   **尺寸结论**：直接按 **1000mm** 执行，不再讨论 `650mm` 极限值。这个尺度才同时容纳机器、窗帘、检修缝和视觉呼吸。
        *   **空间逻辑**：这里是设备位与客厅南端的呼吸边，不再硬塞高柜门，不把整面墙拖成笨重满铺。
        *   **软装**：允许在空调旁补一株大型绿植，遮挡部分管线并把东墙最南端松开。
*   **施工要点**：
    *   **封窗处理**：窗户玻璃务必贴防爆膜，再做假墙遮蔽；柜体与假墙必须控制在同一立面关系，不做额外深色包边和凸出窗套。
    *   **玻璃柜结论**：**不设大面积玻璃展示柜**。东墙的任务是后勤收纳，不是陈列墙。
    *   **开启方式**：主高柜统一采用 **45°背切暗拉手 / 门边内扣手位**，**不用反弹器做主开启方式**。
    *   **门板工艺**：统一平板门，整面墙只保留“暖白大面 + 深色茶咖核”两层关系。
    *   **立面表达**：统一按“满墙高柜 + 中部茶咖功能核 + 南端空调位”描述，不再使用“展示柜”口径。

### 3. 沙发背桌
*   **合理性深度分析 (Rationality Analysis)**：
    *   **结论**：**成立，但前提是把它当成真使用家具，而不是摆件条案。**
    *   **Why Yes (优势)**：
        *   **视觉压舱石**：解决沙发背后空荡荡的不安全感，增加空间的层次感和厚重感。
        *   **交流零阻碍**：工作/喝茶时抬头就能看到电视和家人，避免了传统书房的“面壁思过”和“家庭隔离感”。
        *   **洄游动线**：强制形成“回”字形动线，让气流和人在空间中流动更顺畅。
    *   **Why No (潜在风险与解决方案)**：
        *   *风险*：电源难接。 *解法*：默认按**下沉式隐藏地插**预留；若最终不用地插，也必须保留隐蔽通道，不接受明线压条。
        *   *风险*：显得凌乱。 *解法*：桌子深度控制在 60cm 以内，且必须配备**强收纳功能的各种盒子**（收纳遥控器/数据线）。
*   **布局设计实操 (Layout & Specs)**：
    *   **尺寸黄金法则**：
        *   **长度**：`2400mm` (假设沙发3.2m，两边各留40cm呼吸空间)。
        *   **深度**：`500mm - 550mm` (核心！太宽像餐桌，太窄放不下笔记本。55cm是兼顾书写与美感的极限)。
        *   **高度**：`750mm` (标准书桌高，沙发靠背通常在650-800mm，桌子略低于或齐平于沙发靠背视觉最舒适)。
    *   **选材建议**：
        *   **桌面**：5cm厚 **黑胡桃木直拼板** (大板)，保留木纹质感，边缘做斜切工艺 (显得轻盈)。
        *   **桌腿**：亚克力悬浮腿 (极简) 或 黑色实木片腿 (稳重)。
    *   **配套软装**：
        *   **灯光**：桌面上必须有一盏**有雕塑感的台灯** (如 Flos Snoopy 或 陶瓷底座灯)，作为中景照明。
        *   **座椅**：2把 **昌迪加尔椅 (黑色藤编)** 或 **中古悬臂椅**。平时必须能完全推入桌底，不占过道。
    *   **电力系统 (The Universal Power Strategy)**：
        *   **是否必须供电**：如果背桌承担 **台灯、手机充电、笔记本、临时办公** 中任一功能，就视为**必须供电**；只有当它退化成纯装饰条案时，才可以不留。
        *   **关于“明线+隐藏条”的避坑警告**：坚决**不建议**从墙面拉明线并用隐藏条/压线槽贴在地面。大横厅的核心优势是地面的无界延伸感，隐藏条不仅会像一道疤痕割裂空间，更会在高频的“洄游动线”上制造绊脚隐患，且成为扫地机器人的清洁死角。
        *   **正式方案**：默认采用 **下沉式隐藏地插**；只有在后期明确拒绝机械地插时，才降级为“隐蔽通道 + 小型定制出线盖”。
            *   **方案A：下沉式隐藏地插（默认）**。插头插在凹槽内部，盖板合上后地面依然平整，不怕扫地机碰撞。
            *   **方案B：定制微型出线盖（备选）**。结构里预埋一根 **50mm PVC 通道** 直通墙面，但**完成面只允许看到小型金属出线盖**，绝不能裸露一个“水管口大小”的洞。
            *   **保护区坐标（针对方案A/B）**：**背桌右侧桌腿内侧 20cm，且位于桌板深度中心附近**。
            *   **为什么这是最优解？**
                1.  **全兼容**：无论您买高脚沙发还是落地沙发，这个位置都在桌子下面，互不干扰。
                2.  **防踢保护**：桌腿是天然的挡箭牌。人的脚不会去踢桌腿，形成了“死角保护区”。
                3.  **走线极短**：线出来直接顺着桌腿爬上去，视觉上几乎隐形。
        *   **【关键】坐标落地算法 (The Coordinate Calculation)**：
            *   **痛点**：位置一旦算错，地插就会被压在沙发腿下，或露在外面很难看。所以现在只能先锁**保护区和算法**，不能把毫米数当成最终施工坐标。
            *   **公式**：`地插离电视墙距离` = `最佳观影视距` + `沙发靠背厚度` + `(桌板深度 ÷ 2)`
            *   **实操举例**：
                *   假设电视墙完成面到沙发靠背视距为 **3700mm** (当前方案推荐值)
                *   假设您看中的沙发靠背厚度为 **300mm**
                *   假设书桌深度为 **550mm** (一半即275mm)
                *   **地插中心点位置** = 3700 + 300 + 275 = **离电视墙 4275mm 处**。
            *   **容错技巧 (The Tolerance Trick)**：
                *   **宁深勿浅**：如果不确定沙发具体款式，就把地插位置**往桌子深处多算 10cm**。桌子盖住地插是最好的，露出来是最丑的。
                *   **左右定位**：不要放在正中心！一定要放在**侧边桌腿内侧 20cm**。正中心容易和人腿打架，侧边才是死角。

### 4. 客厅横厅执行定稿 (Final Execution Plan)
*   **空间定性**：**按“大横厅 / 近方横厅”执行，不按普通竖厅处理。**
    *   **原因**：客厅面宽 `7750mm`、进深 `6090mm`，南向采光沿长边展开，且北侧还与中庭形成联动。这里的正确做法是建立**中央家具岛**，而不是把所有家具贴墙排队。
*   **主结构定稿**：固定为 **“西墙主景 + 中部沙发岛 + 沙发背桌功能带 + 东墙后勤界面 + 南北留通道”**。
    *   **否定项**：不再改成传统 L 型围合客厅，不再增加第二组主沙发，不再把东墙当第二主背景。

#### 客厅平面尺寸定稿表 (Living Room Dimension Lock)
*   **电视墙完成面至沙发靠背**：按 **3700mm** 控制。
    *   **结论**：这是 `85` 寸电视与当前客厅尺度下最稳的观影视距，不再来回摇摆。
*   **沙发形式**：**低靠背直排模块沙发**。
    *   **总宽**：建议控制在 **3400-3600mm**。
    *   **总深**：建议控制在 **950-1050mm**。
    *   **方向结论**：**不选厚重 L 型大转角**。横厅的核心价值是岛式与回游，不是把体块向角落堆死。
*   **沙发背桌**：**保留，并按真使用家具执行**。
    *   **长度**：**2400mm**
    *   **深度**：**550mm**
    *   **高度**：**750mm**
    *   **配套座椅**：只放 **2 把**，且平时必须能**完全推进桌底**。
    *   **材质原则**：优先 **黑胡桃木直拼板** 或 **深炭灰木纹**，只做轻体量桌腿，不做边柜化、箱体化。
*   **茶几**：采用 **轻体量圆角茶几**，不做大块厚重方茶几。
    *   **建议尺寸**：主茶几长边控制在 **1200-1400mm**，短边控制在 **650-800mm**。
    *   **逻辑**：客厅已经有背桌承担一部分功能，茶几必须主动退后，避免中部家具岛过满。
*   **东墙后勤界面**：按 **`2.6m` 北段收纳 + `2.4m` 茶咖核 + `1.0m` 南端空调位** 锁死。
    *   **施工红线**：若现场存在尺寸误差，**只允许北段收纳吸收误差**，不动茶咖核，不动空调位。
    *   **否定项**：南端不再补高柜、不做展示柜、不叠加装饰边框。

#### 通道与秩序红线 (Circulation Rules)
*   **背桌至东墙柜体净通道**：尽量保留 **800-900mm**。
    *   **意义**：这是横厅洄游动线的底线。低于这个值，背桌会从“加分项”变成“堵路家具”。
*   **沙发岛南北两端通道**：尽量保留 **1000mm 左右**。
    *   **意义**：给阳台方向与中庭方向都留出顺畅穿行面，维持大平层的松弛感。
*   **中央体块控制原则**：客厅中央已经有 **沙发 + 茶几 + 背桌** 三层关系，**不再增加第四个强中心**。
    *   **否定项**：不再增设第二组主角单椅、大边柜、展示架或厚重雕塑桌。

#### 机电预留锁定项 (Must Lock Before Construction)
*   **电视墙预埋**：按既定方案保留 **50 管预埋**，服务电视、盒子与音响系统。
*   **背桌供电**：既然背桌保留，默认执行 **下沉式隐藏地插** 或 **50mm 隐蔽通道 + 小型出线盖**，不接受明线压条。
*   **沙发侧插座**：两侧插座位必须跟随沙发岛落位同步锁定，不能等家具进场后再想办法补。
*   **茶咖区电源**：按东墙 `2.4m` 茶咖核中心布置，不向两侧扩散零碎插座。

#### 软装落地口径 (Soft Furnishing Rule)
*   **沙发颜色**：固定在 **米灰 / 大象灰** 家族，优先哑光布艺、磨砂绒或高支棉麻。
    *   **否定项**：不选亮皮、不选焦糖棕大体块、不选高反光材质。
*   **地毯**：建议使用 **浅燕麦 / 米白** 大地毯，作为中央家具岛的柔化层。
    *   **尺寸建议**：优先按 **3000×4000mm** 级别理解，宁大勿小，避免沙发岛漂在亮光砖上。
*   **绿植**：允许保留 **1 株高大绿植** 作为垂直点睛，优先放在东墙空调位旁或电视柜侧边。

#### 最终结论
*   **这套客厅的执行方向不是“再加东西”，而是“把横厅秩序锁死”。**
*   **后续所有家具下单、插座放线、地插预埋与软装选型，均以本节口径为准，不再回退到普通客厅思路。**

## A.2 主卧区域 (Master Bedroom)
*   **面积**：26.70m² (3.57m × 7.48m)

### 布局细节 (Master Plan for Master Bedroom)

1.  **东墙：7m 极简收纳与梳妆系统 (East Wall System)**
    *   **总原则**：**大面必须封闭，深色只集中在梳妆位，不设玻璃展示柜，不做花哨门型。**
    *   **口径修正**：**东墙总长接近 7m，不等于 7m 都应该做成满深柜体。** 原阳台侧只要条件不足，就应该主动退成浅收口或白墙，而不是硬把柜子拖到底。
    *   **尺寸定案**：按 **A区 1.0m + B区 3.2m + C区 1.6m + D区 1.2m = 7.0m** 执行；若现场总长有误差，**只允许 B区吸收误差**，不要动 A/C/D 的功能尺寸。
    *   **A区 (入口 1.0m)**：**纯封闭收纳**。
        *   **门板拆分**：建议做 **2 扇 500mm 等宽高门**，与后续主力柜门缝统一。
        *   **功能定位**：放行李、换季被芯、低频杂物，保证入门第一眼安静。
    *   **B区 (中段主力衣柜 3.2m)**：**一门到顶 PET 肤感白主收纳**。
        *   **门板拆分**：建议做 **6 扇等宽门**，单扇控制在 **530-535mm 左右**；这是 3米级高柜门稳定性和立面节奏的平衡点。
        *   **门板限制**：单扇净宽尽量 **不超过 550mm**，超过这个值，3米级高柜门后期更容易变形、下坠、手感发飘。
        *   **内部逻辑**：按 **2 组长衣 + 2 组短衣 + 2 组叠放 / 抽屉** 执行，不在外立面暴露任何开放格。
        *   **从北到南 6 模块定案**：
            *   **B1 (约 530mm)**：长衣区。净挂高建议 **1350-1400mm**，上方保留顶层被褥格。
            *   **B2 (约 530mm)**：长衣区。与 B1 成对，承担大衣、连衣裙、长外套。
            *   **B3 (约 530mm)**：短衣区。上部双层短衣挂放，下部 **3 抽**，放内搭与高频贴身衣物。
            *   **B4 (约 530mm)**：短衣区。上部双层短衣挂放，下部可做 **裤架 + 1 抽** 或层板叠放。
            *   **B5 (约 530mm)**：叠放区。建议做 **4-5 层可调层板**，放针织、家居服、包件收纳盒。
            *   **B6 (约 530mm)**：抽屉 / 配件区。建议做 **4 抽 + 顶部层板**，放首饰盒、腰带、丝巾、小件电器。
        *   **使用逻辑**：靠近入口的 B1/B2 负责大件挂放；靠近梳妆位的 B5/B6 负责叠放与配件，方便更衣和梳妆形成一条连续动作线。
    *   **C区 (中段 1.6m)**：**梳妆 / 阅读位**。
        *   **形式**：做成 **内嵌深色盒子**，而不是在白柜里硬挖一个桌子洞。
        *   **台面建议**：盒子总宽按 **1600mm** 控制，台面净宽控制在 **1500-1550mm**，深度 **520-550mm**，完成面高度 **750mm**。
        *   **坐姿净空**：膝部净宽建议 **900-1000mm**，净高 **620mm 以上**；抽屉只允许做 **单侧抽屉塔**，不要双侧夹住坐姿。
        *   **抽屉塔朝向**：固定放在 **北侧**，靠近主力衣柜；南侧保持打开，把光线和坐姿呼吸感留给原阳台方向。
        *   **抽屉塔尺寸**：宽度建议 **350-400mm**，抽屉 **3 层** 即可；不要做落地厚柜，维持悬浮感。
        *   **镜面建议**：采用 **650×900mm 竖向圆角镜** 或同尺度壁挂镜，镜底离台面约 **150-180mm**，不使用自带环形灯的网红镜。
        *   **功能边界**：这里是梳妆和短时阅读位，不是第二个办公桌，不摆大显示器，不做满墙开放书架。
    *   **D区 (原阳台侧过渡带 1.2m)**：**取消满高柜设定**，改为 **浅层收口 / 轻功能带**。
        *   **结论**：这一区域如果实际墙体只有 **250mm 左右有效深度**，就已经不具备做标准衣柜的基础。继续按满高柜硬做，只会得到一段又浅又假、还会挡光的失败柜体。
        *   **宽度判断**：这一区域按 **1.2m 左右** 的残余墙段理解最合理，已经足够做出完整退台，但不值得继续延伸高柜。
        *   **推荐做法**：优先做 **暖白同墙色浅收口带**，深度控制在 **220-250mm**，视觉上当作墙的一部分处理，而不是独立高柜。
        *   **功能选择**：只承担 **薄层收纳 / 展示落点 / 小件托盘台面** 这类轻功能；不承担挂衣、囤货、高频储物。
        *   **更优落地**：建议做 **1000-1100mm 宽 × 220-250mm 深 × 720-750mm 高** 的悬浮浅台面或薄抽屉边柜，两侧保留 **50-100mm** 呼吸边，上方保持白墙或同色护面，别再做通顶高门。
    *   **统一细部**：
        *   **玻璃柜结论**：**不设玻璃柜**。卧室这面柜墙正对床，玻璃门只会带来反光、积灰和视觉噪音。
        *   **拉手结论**：**不用反弹器做主开启方式**。高柜日常使用频率高，反弹器后期故障率和手感都不值得赌。
        *   **开启方式**：主高柜统一采用 **45°背切暗拉手 / 门边内扣手位**；梳妆抽屉采用底部暗扣手，不做明装拉手。
        *   **门板工艺**：统一 **平板门**，不做框线门、格栅门、长虹玻璃门、铝框玻璃门。
        *   **踢脚**：统一 **80mm 哑光黑金属内缩踢脚**，做轻微悬浮感，但不夸张抬高。

2.  **西墙：3100mm 平直背景墙与睡眠核心 (The Calm Bed Wall)**
    *   **总尺寸口径**：按 **7.00m 有效设计长度**控制，拆分为 **1.10m 北侧留白 + 3.10m 床背景主体 + 2.80m 南侧延展休闲段**。
    *   **锚定原则**：**床本体**才是定位锚点，**床头柜只是跟随构件**，不反向决定背景墙位置。
        *   **床位锚点**：按 **2.0m 床本体**计算，床本体北侧边缘距北墙 **1.65m**，床本体中心线距北墙 **2.65m**。
        *   **背景墙位置**：围绕床本体中心线对称展开，北侧起点距北墙 **1.10m**，南侧终点距北墙 **4.20m**，总长 **3.10m**。
        *   **床组校核**：按 **双床头柜各 500mm** 计算，床组整体最北端距北墙约 **1.15m**，最南端距北墙约 **4.15m**，背景墙每侧保留约 **50mm 呼吸边**。
    *   **北侧留白区 (1.10m)**：**纯白缓冲，不做装饰动作**。
        *   **材质**：**纯白哑光乳胶漆**（#F5F5F5 / NCS S 0502-Y），**不做格栅、不做护墙线、不做木饰起势**。
        *   **家具策略**：默认保持留白；若现场觉得太空，只允许放 **1200×300×750mm 极窄条案** 或 **单株竖向绿植**，二选一。
        *   **机电**：条案位预留 **双五孔 + Type-C** 插座；开关面板与床区场景联动。
    *   **床背景主体 (3.10m)**：**满高、平直、薄构件，不做整面西墙满铺**。
        *   **构造厚度**：完成面厚度控制在 **35-40mm**，只承担找平、灯带和完整收口，不做厚重舞台体块。
        *   **主体材质**：**暖灰木饰面 / 烟熏浅胡桃木**，纹理顺直、色温偏暖，避免深胡桃压低卧室情绪。
        *   **软包核心区**：居中布置 **2350mm 宽 × 1180mm 高** 硬包，离地 **420mm** 起，顶部结束于约 **1600mm** 高度。
            *   *材质*：微纹理皮革或高耐磨布艺硬包。
            *   *颜色*：**燕麦灰 / 暖灰褐 / 灰咖**，以 **燕麦灰** 为默认落地色。
        *   **收口细节**：背景墙四周统一留 **8mm 阴缝**，颜色用深灰褐，不上金属条，不做 L 型折边。
    *   **南侧延展休闲段 (2.80m)**：**床区退后，休闲区展开**。
        *   **余量口径**：按 **床组整体最南端距北墙 4.15m** 计算，南侧到 7.00m 设计终点仍保留约 **2.85m**；若只按 **床本体** 计算，南侧余量约 **3.35m**。
        *   **西南角设备位**：立式空调固定在 **最西南侧**，设备位按 **500-550mm** 宽度控制，外侧额外预留 **150-200mm** 回风与检修缝，不再叠加柜体。
        *   **休闲位尺度**：南侧偏中或偏东放 **轻体量懒人椅 / 休闲单椅 + 小边几**。
            *   *单椅建议*：宽 **800-850mm**、深 **850-950mm**、座高约 **380-420mm**。
            *   *边几建议*：直径 **350-400mm** 即可，不上大茶几，不做第二组家具。
            *   *朝向建议*：以 **朝北或朝西北** 为主，让坐姿视线回看床区和南向窗景，不斜冲空调。
        *   **通行红线**：单椅与空调之间、单椅与最外侧窗帘之间，均尽量保留 **800mm 左右** 的净活动带，避免南端被家具堆成死角。
        *   **空间效果**：这里是床区向阳台的自然延展，保持白墙与轻体量家具的逻辑，避免把南端拖回“背景墙尾巴”。
    *   **施工交底尺寸清单 (North-to-South Control Line)**：
        *   **放线基准**：统一以 **北墙完成面** 作为 `0` 点，所有西墙落位均从北向南放线。
        *   **背景墙起点**：`1100mm`。
        *   **床组整体最北端**：`1150mm`（按双床头柜各 `500mm` 校核）。
        *   **床本体北侧边缘**：`1650mm`（**床的正式锚点**）。
        *   **床本体中心线**：`2650mm`。
        *   **床本体南侧边缘**：`3650mm`。
        *   **床组整体最南端**：`4150mm`。
        *   **背景墙终点**：`4200mm`。
        *   **空调设备带起点**：建议控制在 `6200mm` 以后，确保床区与休闲位之间先保留完整的停留面。
        *   **7.00m 设计终点**：西墙南端休闲区收口线；此点之后由最外侧窗帘与南向采光面接管，不再延续背景墙语言。

3.  **南向打通界面与空调 (Integrated South Edge & AC)**
    *   **结构**：主卧与阳台区**直接打通**，按一个完整房间处理。
    *   **地面**：与主卧同材连续通铺，强化一体感。
    *   **空调**：立式空调固定在 **最西南角**，优先弱化设备存在感，不与床背景抢主角。
    *   **休闲位**：南端留给 **懒人椅 / 休闲单椅**，作为睡眠区向窗边的过渡，不承担第二功能区任务；这里做的是“能坐下来的留白”，不是第二个小客厅。
    *   **窗帘**：安装在 **最外侧南向窗面**，作为同一套窗帘系统处理。

4.  **北侧入户：留白缓冲区**
    *   **定位**：**视线缓冲优先**，不是第二功能区。
    *   **材质**：**纯白哑光乳胶漆**，保持轻盈与通透，拒绝格栅、护墙板、装饰框等“提醒你这里被设计过”的做法。
    *   **基层找平 (关键工艺)**：**不做木工基层**，采用 **油工冲筋找平**，确保白墙平整如镜。
    *   **家具选型**：默认留白；若需要落点，仅允许 **极窄条案**（1200×300×750mm）或 **单株竖向绿植**。
    *   **线性边界**：通过 **1.10m 白墙留白 → 3.10m 背景墙主体** 的切换建立秩序，不再依赖睡眠岛折边和格栅起势。

## A.3 多功能中庭 (Central Atrium)
*   **总面积**：38.53 m² (9.24m × 4.17m)
*   **设计决策 (Crucial Decision)**：**保持完全开放，保留极致大平层体验**。
    *   *获益*：彻底释放 9.24米 的超大横向面宽，形成 **“南客厅(47m²) + 中庭(38m²)”** 的双巨厅格局。这种空间尺度在豪宅中都极为罕见。

### 1. 空间功能重构 (Open Layout)
整个中庭通过家具和功能分区定义空间：

*   **Zone 1: 西侧·端景与交通轴**
    *   **位置**：连接客厅北门、卫生间与北向房间的交通枢纽。
    *   **视觉重心**：
        *   **北墙 (The Focal Wall)**：从客厅看出来的第一眼。悬挂巨幅抽象画 (150cm x 180cm) 或设置端景台。
        *   **灯光**：偏光洗墙灯均匀照亮北墙，营造安静、克制的端景氛围。
*   **Zone 2: 东侧·轻运动与阅读区 (Wellness & Lounge)**
    *   **位置**：靠东窗。
    *   **布局建议**：
        *   **跑步机/划船机**：直接面向东窗摆放。运动时视野开阔，不再憋屈在柜子里。
        *   **懒人沙发/单人椅**：配合落地灯，打造一个开放式的阅读角或发呆区。
        *   **极简收纳**：西墙低矮长条柜，用于收纳瑜伽垫、哑铃等小件器材，台面可放置绿植。

### 2. 氛围与材质 (Atmosphere & Materials)
*   **地面**：通铺木地板/地砖，与客厅保持一致，强化延伸感。
*   **墙面**：全屋统一的暖白/浅灰基调，仅在端景墙做材质跳跃。
*   **天花**：**极简大平顶**。采用无主灯设计，两条超长磁吸轨道贯穿东西（9米长），拉伸视觉张力。

### 3. 中庭空间价值
这个 38m² 的中庭不是过道，而是全屋的第二公共核心。它的价值在于：

*   **空间升级**：中庭作为可以停留、活动、展示的厅，而不是通行通道。
*   **采光延展**：东侧采光能够一直延展到西侧服务带，西侧入口区域不会陷入昏暗。
*   **活动尊严**：运动器材和阅读位直接面对采光面，使用体验更完整。
*   **社交溢出**：当客厅承载人数上来时，中庭自然承担第二停留场。

### 4. 软装陈列方案：如何填满空间而不拥挤？
我们用 **“三个视觉岛屿”** 来破解“空”的焦虑，而不是用墙把它堵死：

1.  **东岛 (入户端景)**：
    *   **配置**：**极简玄关条案 (1.5m长)** + **抽象挂画** + **台灯**。
    *   *作用*：一进门（或从客厅出来）看到的不是空墙，而是清晰的端景艺术角。随手放钥匙、香薰。
2.  **中岛 (流动区)**：
    *   **配置**：**留白**。
    *   *作用*：给扫地机器人、给未来的孩子/宠物、给您自己留出一条宽敞的“跑道”。在家里能走直线，是一种极大的心理享受。
3.  **东岛 (活动区)**：
    *   **配置**：**跑步机 (面向窗)** + **单人沙发/躺椅 (面向中庭)** + **落地钓鱼灯** + **大盆绿植 (琴叶榕/龟背竹)**。
    *   *作用*：这组家具本身就是雕塑。它们占据了东侧 15m² 的空间，让空间看起来丰富且有生活气息。

## A.5 专项施工：客厅悬浮电视背景墙

### 1. 85寸电视悬浮墙施工工艺
*   **结构安全**：
    *   **双面背对背结构 (Back-to-Back System)**：
        *   **A面 (主卧侧)**：标准衣柜结构，需升级为 **18mm厚背板** 以增强整体刚性。
        *   **B面 (客厅侧)**：在衣柜背板上打 **30mm-50mm 木龙骨/欧松板基层**。
            *   *作用*：走电线管、安装插座底盒、找平、制造悬浮所需的底部内收空间。
    *   **侧面收口 (The Edge)**：衣柜侧板与背景墙侧边必须使用 **同材质见光板** 或 **金属收口条** 统一包覆，形成完整的 650-700mm 建筑体块，杜绝拼接感。
    *   **隔音处理**：在衣柜背板与客厅背景墙基层之间的空隙填充 **高密度隔音棉**。
    *   砖墙方案：轻钢龙骨垫出造型层，挂架穿过造型层打入砖墙
    *   钢架方案：挂架必须锁在内部方管上，严禁只锁在木板上
*   **悬浮做法**：
    *   底部悬空：背景板离地20cm
    *   内凹踢脚：墙根向内收5cm做黑色不锈钢踢脚
    *   灯光槽：底部预留2cm宽×1cm深铝槽，嵌入3000K COB灯带，柔光罩见光不见灯
