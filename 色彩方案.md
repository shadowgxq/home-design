# 装修色彩与材质规范 (Color & Material Specs) - Plan C: 极简·高级灰 (Advanced Grey)

> **核心逻辑流**: 地面(暖灰) → 墙面(菱花白) → 骨架(深灰) → 柜体(暖白+烟熏木)。


## 1. 第一层：天地基调 (The Foundation)
> **目标**: 打造一个干净、明亮且有温度的“容器”。

| 选项 | 色号/特征 | 视觉感受 | 与“亮光砖+灰门”的匹配度 |
| :--- | :--- | :--- | :--- |
| **菱花白 (Master Choice)** | **OW038-4** (带极少黄+灰) | **羊脂玉般的温润**。不像纯白那么冷，也不像奶油色那么黄。 

| 对象 | 材质/工艺 | 颜色/色号 | 核心解析 (Why?) |
| :--- | :--- | :--- | :--- |
| **地面** (Ground) | **超平釉 / 全抛釉 (亮光)** | **暖灰色 (Warm Grey)** | **核心基石**。必须是 **亮光 (Glossy)**。带一点咖色调的暖灰。<br>**理由**: 亮光砖能最大化反射光线，营造豪宅的通透感与奢华感。拒绝柔光砖的渗污问题，也拒绝微水泥的暗沉。<br>**美缝**: 聚脲调色，哑光，做到“缝隙消失”。<br><br>⚠️ **选购避坑指南 (Buying Guide)**:<br>1. **看灯影 (The Reflection Test)**: 站在侧面看灯光倒影。如果是标准的圆形/直线，说明平整度好（超平釉）；如果像哈哈镜一样扭曲（水波纹严重），坚决不要。<br>2. **定色调 (The Warmth Check)**: 拿一张 A4 白纸放在砖上对比。必须选 **“带咖色/米色倾向的灰”**。如果对比之下显蓝、显青，那就是冷灰，会毁了全屋温馨感。<br>3. **审纹理 (The Vein Check)**: 拒绝“大花白”、“黑白根”等强纹理。要选 **“若隐若现的丝绸纹”** 或 **“极简微水泥纹的亮光版”**。铺贴后远看是纯色，近看有细节，才是高级。<br> 大平层，** 750×1500mm**
| **墙面** (Wall) | **乳胶漆 (全哑光)** | **菱花白 (OW038-4)** | **包容**。比纯白更柔，能中和亮光砖的冷。<br>**工艺**: 滚涂施工，保留微颗粒质感，防止墙面反光。 |
| **顶面** (Ceiling) | **乳胶漆 (全哑光)** | **菱花白 (OW038-4)** | **延伸**。墙顶同色，模糊边界，显层高。 |

## 2. 第二层：骨架与立面系统 (The Skeleton & Facade)
> **目标**: 解决“明装踢脚线、黑框玻璃门、卧室木门”的材质收口与色彩逻辑，确保空间连贯不割裂。

| 对象 | 材质/工艺 | 颜色/色号 | 核心解析 (Why?) |
| :--- | :--- | :--- | :--- |
| **踢脚线** (Skirting) | **明装超薄铝合金** (厚1.2-3mm) | **哑光黑 (Matte Black)** <br>**理由**: 相比深灰色，哑光黑的线条感更强，能像画框一样清晰勾勒出空间轮廓。它能与公共区黑框门、黑色开关面板呼应，形成清晰的金属边界系统。<br>**注意**: 这里的黑色优先服务 **金属线条**，不要求和卧室木门做成同材同色。 |
| **木门及门套** (Wood Door)| **实心平板门+木质极窄套** | **深炭灰开放漆木纹 (Charcoal Wood Grain)** | **柔和的酷感**。如果觉得纯黑太硬，**深炭灰是最佳替代**。<br>它依然能压住亮光地砖的“飘”，但视觉上更温润。这种深灰木纹在光线下非常有质感，是意式极简的标配。它属于 **木质体块系统**，与黑色金属边框同属深色家族，但不要求完全同色。 |
| **客厅阳台玻璃门** (Glass Door) | **极窄铝合金** | **黑色边框 + 灰色玻璃** | **呼应**。与黑色的踢脚线、卧室门完美呼应，形成全屋统一的“黑色骨架系统”。 |
| **窗框/厨房移门** (Windows) | **极窄金属** | **黑色 / 深灰色** | **呼应**。与客厅阳台的黑框玻璃门形成呼应系统。 |

## 3. 第三层：收纳体块 (The Body)
> **目标**: 柜子即墙体。大面积留白，局部深色点缀，呼应骨架。

| 对象 | 材质/工艺 | 颜色/色号 | 核心解析 (Why?) |
| :--- | :--- | :--- | :--- |
| **柜门大面** (Main) | **PET 肤感膜** | **全屋羊脂白 (Warm White)** <br>- **避坑**: 浅暖灰柜体很难选，容易跟地砖和墙面撞色。白色永远不会错。 |
| **局部跳色/电视柜** (Accent) | **深炭灰木纹 / 深灰岩板** | **深炭灰 (Charcoal Grey)** | **全屋统一**。电视柜与局部功能核心建议统一在 **深炭灰木纹 / 深灰岩板家族**，但**不必与踢脚线、金属边框完全同色同材**。<br>1. **材质呼应**: 悬浮电视柜建议选与卧室门同款的 **深炭灰木纹板**。<br>2. **色彩闭环**: 家里形成“三层关系”——地面的暖灰、墙柜的白、**黑色金属线条 + 深炭灰木质体块**。 |
| **柜体五金** | **免拉手 / 45°背切暗拉手** | **N/A** | **极简，但要耐用**。卧室高柜**不建议把反弹器当主开启方式**，高门板时间一长手感会变差、对缝也容易乱。默认用 **45°背切暗拉手或门边内扣手位**，只在小抽屉上做暗扣手。 |

### A. 客厅满墙高柜与茶咖操作区 (6米长，3米高)
*   **核心逻辑**: **浅色大面退后，深色功能核心前置**。这面墙的任务是收纳和小家电操作，不是做第二个主秀场。
*   **尺寸定案**: 按 **2.6m 北段封闭收纳 + 2.4m 中部茶咖核 + 1.0m 南端空调位** 理解；若现场存在尺寸误差，只由北段封闭收纳吸收。
*   **尺寸与材质排布 (从左至右)**:
    1.  **北段封闭收纳 (约 2.6m)**: **羊脂白 PET 封闭门**。建议拆成 **5 扇约 520mm** 宽的高门，负责囤货、器皿和低频小家电，让柜体像墙而不是像陈列架。
    2.  **中部茶咖功能核 (约 2.4m)**: **深炭灰木纹** 只集中在背板与操作台。
    3.  **上下柜体**: 上下柜门继续使用羊脂白柜门，让深色仅集中在功能核心，不向整面高柜扩散。
    4.  **南端空调机位 (约 1.0m)**: 保持留空，预留给立式空调柜机与窗帘位；允许大型绿植介入，但不再做高门收尾。
*   **客厅柜体专属细节结论**:
    *   **玻璃结论**: **不做大面积玻璃展示柜**，电视墙两侧也不默认进入玻璃陈列语言。
    *   **拉手结论**: 主高柜统一采用 **45°背切暗拉手 / 门边内扣手位**，**不采用反弹器作为主开启方式**。
    *   **深色控制**: 深炭灰木纹只留在茶咖核内部，不扩散到整段高柜门，不额外引入第三种主木色。
### B. 主卧 7m 极简收纳与梳妆系统 (The East Wall System)
*   **核心逻辑**: “封闭退后”与“局部深色聚焦”的穿插。整排柜体以安静、封闭和耐看为第一原则。
*   **尺寸定案 (从北向南 / 从入口向阳台)**: **1.0m 入口收纳 + 3.2m 主力衣柜 + 1.6m 梳妆位 + 1.2m 原阳台侧退台**。若现场尺寸有微误差，由主力衣柜段吸收。
*   **尺寸与材质排布 (从北向南 / 从入口向阳台)**:
    1.  **A区 - 入口收纳区 (约 1.0m)**: **羊脂白 PET 封闭门**。入口保持安静的封闭收纳。
    2.  **B区 - 主力衣柜区 (约 3.2m)**: **羊脂白 PET 封闭门**。正对床头，大面积白色让空间不显拥挤。底部搭配 **哑光黑金属踢脚线**，形成轻微的悬浮错觉。
        *   **门板选择**: 统一 **平板门**，建议拆为 **6 扇约 530-535mm 宽** 的高门；高门背面加直器，避免后期变形。
        *   **玻璃结论**: **不做玻璃柜**。主卧正对床位，任何玻璃门都会让衣物和反光成为视觉污染。
        *   **内部收纳结构**: 从北到南按 **长衣、长衣、短衣+抽屉、短衣+裤架、叠放层板、配件抽屉** 六模块展开，让更衣逻辑从入口一路过渡到梳妆位。
    3.  **C区 - 内嵌式梳妆台/阅读角 (约 1.6m)**:
        *   **形式**: 挖空柜体，做成内嵌的悬浮盒子。
        *   **材质/撞色**: **深炭灰木纹 (Charcoal Wood)**。与卧室门和客厅局部深色体块统一，但只限于梳妆位盒子内部，不向白柜大面扩散。
        *   **配置**: **深炭灰木纹台面** + **单侧悬浮抽屉塔** + **竖向圆角镜** + 内部 4000K 隐形补光。
        *   **尺寸建议**: 盒子总宽 **1600mm**，台面净宽 **1500-1550mm**、深 **520-550mm**、高 **750mm**；镜面建议 **650×900mm**，不要环形灯镜。
        *   **朝向建议**: 抽屉塔固定在 **北侧**，把南侧留空给坐姿和采光，避免梳妆位在临窗方向变闷。
    4.  **D区 - 原阳台侧过渡带 (约 1.2m)**: **不再做满高柜**。如果现场只有 **250mm 左右有效深度**，就改做 **暖白浅收口带 / 薄台面**，让柜体在这里自然退场，把光线还给南端。
        *   **形式建议**: 采用 **1000-1100mm 宽、220-250mm 深、720-750mm 高** 的浅台面、薄抽屉边柜或纯白收口墙面，不再继续延伸通顶高门。
        *   **颜色策略**: 与白柜和墙面保持同一暖白体系，故意弱化存在感，不再加深色体块。
*   **东墙专属细节结论**:
    *   **颜色**: 主体只保留 **羊脂白 + 深炭灰木纹** 两层关系，不再引入第三种木色。
    *   **拉手**: 主高柜用 **45°背切暗拉手 / 门边内扣手位**，**不采用反弹器作为主开启方式**。
    *   **开放格**: 除梳妆位内部，不再增加开放格和玻璃门。
    *   **门缝控制**: 垂直缝建议控制在 **2.5-3mm**，所有高门必须对齐，不允许忽宽忽窄。
### C. 主卧西墙 3100mm 床背景墙 (The Bed Wall)
*   **核心逻辑**: **床区集中发力，北侧和南侧主动退后**。这面墙的任务是托住 2.0m 床，不是把整面西墙都变成重造型。
*   **材质排布 (从里到外)**:
    1.  **底墙**: **暖白哑光乳胶漆**。北侧与南侧白墙保持纯净，不做格栅、不做装饰框。
    2.  **主背景板 (3100mm)**: **暖灰木饰面 / 烟熏浅胡桃木**。做成满高、平直、35-40mm 厚的完整构件。
    3.  **软包核心区**: **燕麦灰 / 暖灰褐** 硬包。尺寸控制在 **约 2350mm 宽 × 1180mm 高**，让触感与睡眠氛围集中在床区本体。
    4.  **收口**: 四周统一 **8mm 阴缝**，颜色用深灰褐，杜绝金属条和厚折边。
### D. 总结
*   **客厅西墙**: **悬浮电视柜** (深炭灰木纹) + **背景墙** (羊脂白/岩板)，形成经典的“深浅对比”；两侧只做收边，不走玻璃展示柜路线。
*   **客厅东墙**: **羊脂白** (大面) + **深炭灰木纹** (中部茶咖功能核)，呼应西墙电视柜，南端留给 `1000mm` 空调位与呼吸边。
*   **主卧大衣柜**: **羊脂白** (大面) + **深炭灰木纹** (内嵌梳妆台) + **哑光黑金属踢脚线**，坚持全封闭平板门，不做玻璃展示柜。
*   **主卧西墙床背景**: **暖灰木饰面** (主背景板) + **燕麦灰硬包** (睡眠触感核心) + **暖白留白墙面** (北侧与南侧退后)。

## 4. 第四层：细节五金 (The Details)
> **目标**: 在“近距离接触”时，感受到精致的工业美学。

| 对象 | 材质/工艺 | 颜色/色号 | 核心解析 (Why?) |
| :--- | :--- | :--- | :--- |
| **门锁/合页** | **磁吸静音锁** | **枪灰色 (Gunmetal)** | **升级**。比黑色更细腻，有金属光泽但不过分耀眼，与灰门绝配。 |
| **开关插座** | **哑光面板** | **深灰/银灰 或 纯白** | **分区域策略 (极简核心：隐形或点缀)**：<br>1. **菱花白大白墙上**：建议选**纯白或极浅灰 (哑光)**，让面板“隐入墙体”，避免墙面出现“打补丁”的突兀感。<br>2. **深色背景 (如黑白电视墙的黑色区、木饰面、深灰硬包)**：必须选**深灰色 (Deep Grey) 或 哑光黑**，与背景色融为一体。<br>*(注：高端做法是根据墙面底色买两种颜色的面板混搭，坚决不用刺眼的亮面金/亮面银)* |
| **灯具外壳** | **防眩筒灯/磁吸轨** | **哑光黑 / 深灰** | **点缀**。天花板上的灯具依然可以选黑色，作为空间的最高音，增加层次。 |

## 5. 第五层：软装陈设 (The Soft Touch)
> **目标**: **救命稻草**。必须用软装来中和“亮光砖+深灰骨架”的冷硬感，注入“家的温度”。

| 对象 | 材质 | 颜色建议 | 核心解析 (Why?) |
| :--- | :--- | :--- | :--- |
| **沙发** (Sofa) | **磨砂绒 / 高支棉麻 / 科技布** | **米灰色 (Greige) / 大象灰** | **高级且耐看**。既然觉得焦糖色太“重”，那就回归 **同色系极简**。<br>1. **逻辑**: 沙发颜色比地砖深 2-3 度，比墙面深 4-5 度。形成“墙浅-地中-沙发深”的自然过渡。<br>2. **质感**: 放弃光面皮，改用 **哑光磨砂绒** 或 **粗织棉麻**。这种“毛茸茸”的质感能极大缓解亮光地砖的冰冷，让家更像家，而不是样板间。<br>3. **提气**: 觉得太素？扔两个 **深海蓝** 或 **酒红色** 的丝绒抱枕，瞬间点亮。 |
| **地毯** (Carpet) | **羊毛/混纺短绒** | **米白 / 浅燕麦色** | **提亮/柔化**。既然地面是冷硬的亮光灰砖，地毯就要选 **浅色、暖色、毛茸茸** 的。它像一个温暖的岛屿，漂浮在冰冷的湖面上。 |
| **窗帘** (Curtain) | **高精密 + 幻影纱** | **奶咖色 / 燕麦色** | **滤镜**。白天全屋挂纱，让光线漫反射。窗帘布选 **奶咖色**，比墙面深一点的暖色，给空间加一层“暖色滤镜”。 |
| **绿植** (Plants) | **大叶植物 (天堂鸟/龟背竹)** | **鲜绿色** | **生命力**。在深炭灰的电视柜旁，必须放一盆高大的绿植。绿色的生机与深灰的冷峻是绝配。 |

## 7. 色彩方案总结
想象一下：
你走进家门，地面是 **光洁如镜** 的 **暖灰色亮光砖**，倒映着天花板的灯光，满屋通透奢阔。
墙面是柔和的 **菱花白**，平衡了地面的冷硬。
所有的深色元素分成两套系统：**黑色金属线条**（踢脚线、公共区门框、局部五金）与 **深炭灰木质体块**（卧室门、悬浮电视柜、局部柜体跳色）。两者同属深色家族，但各司其职，构成更清晰的极简骨架。
**但是，这里并不冰冷。**
客厅中央，一组 **米灰色或大象灰沙发** 承接地面与墙面的过渡，脚下是 **米白色的羊毛地毯**，柔软而亲肤。
客厅东墙的 **深炭灰木纹茶咖操作区** 在暖白满墙高柜中形成清晰的功能核心。
一株高大的 **天堂鸟** 在电视柜旁舒展着绿叶。

**这就是 Plan C (Warm Luxury)：硬装高冷如绅士，软装温情如拥抱。这才是高级且好住的家。**
