# 装修色彩与材质方案 (Color & Material Scheme)

本方案基于极简风 (Minimalist Style) 设定，强调“通刷”、“顺色”与“肤感”，打造高级且耐看的居住空间。

## 核心配色逻辑
*   **基调 (60%)**: 菱花白 + 暖灰 (墙地面大面积铺陈，奠定温暖且干净的底色)
*   **融合 (30%)**: 羊脂白 + 木纹 (柜体与房门融入背景，木纹增加温润感)
*   **点缀 (10%)**: 哑光黑 + 燕麦/大象灰 (通过家具与软装提升层次，黑色线条勾勒极简感)

## 补充说明
1.  **菱花白 (OW038-4)**: 这是一个非常经典的网红色号，属于暖白偏一点点灰/粉，非常温馨且不显脏，比纯白更有质感。
2.  **美缝“宁深勿浅”**: 这是一个关键施工细节。浅色美缝容易氧化变黄变黑，深色美缝（接近砖色但稍深）能形成阴影效果，视觉上“隐形”，更显整体。
3.  **灯光色温**:
    *   **4000K (自然白)**: 更加明亮清爽，最接近自然光，适合极简风的冷冽感。
    *   **3500K (暖白)**: 如果担心4000K太冷，可选3500K作为折中方案，既有温馨感又不失明亮。
4.  **大板茶桌避坑指南**:
    *   **拒绝亮光漆**: 亮光漆会让大板显得像“暴发户”风格，与极简风格格不入。必须要求厂家做**木蜡油**或**全哑光开放漆**。
    *   **拒绝红木色**: 颜色必须去红，往“黑/灰/棕”方向靠，才能压得住空间。
    *   **拒绝笨重桌腿**: 亚克力腿能制造悬浮感，是极简风首选；或者用极细的黑色哑光金属腿。
