## 2. 改造重点 (标红项)
*   **客厅北门 (连接多功能室)**: 宽度由 800mm -> 2000mm，高度 -> 2400mm (无过梁限制，放心拆改)。
*   **客厅南门 (连接阳台)**: 宽度 -> 3000mm，高度 -> 2400mm (无过梁限制，放心拆改)。
*   **电竞房门**: 
    *   **高度**: 提升至 **2400mm**，与全屋门洞齐平。
    *   **宽度**: 提升至 **2000mm**，与客厅北门对齐。
    *   **结构**: 采用 **“两侧砌实墙 + 中间装移门”** 方案。

## 3. 方案分析与优化建议 (专家诊断结论 - V2版)

### 3.1 关键前提确认
*   **层高优势**：**吊顶后净层高 3.0米** (极度稀缺的大宅层高)。
*   **结构条件**：**不需考虑结构过梁限制** (Owner Specified)。
*   **设计原则**：基于3米净高，普通2.1米门会显得极其小气。**全屋门洞高度建议统一提升至 2.4米**，打造“顶天立地”的豪宅感。

### 3.3 深度解析与修正建议

#### A. 客厅北门 (连接多功能室)
*   **现状**: 2000×2400
*   **评价**: **完美**。3米净高下，2.4米门洞是标准配置。
*   **修正方案**: 保持 2000×2400 尺寸。推荐 **极窄黑色三联动推拉门**，平时常开，形成半开放空间。
*   **玻璃**: **透明灰玻** (通透感)。

#### B. 客厅南门 (连接阳台)
*   **现状**: 3000×2400
*   **评价**: **完美**。
*   **修正方案**:
    *   **尺寸**: 保持 3000×2400。
    *   **形式**: 推荐 **重型提升推拉门 (两轨两扇)**，单扇1.5米宽，视野无敌。
*   **玻璃**: **透明灰玻** (滤光)。

#### C. 电竞房门 (多功能室内部隔断)
*   **现状**: 2000×2400 (吊轨推拉)
*   **最终方案 (Final Decision)**: **非对称实墙 + 侧边悬浮玻璃展示柜 (The Gallery Plus)**。
    *   **高度决策**: **必须 2400mm (与客厅门齐平)**。
        *   **禁止**: 绝对不能做 2100mm。在 3m 层高下，2.1m 的门会显得极其小气，且与外面的 2.4m 客厅门形成“高低眉”，破坏画廊的透视感。
        *   **建议**: 全屋卧室门也尽量提升至 2.4m；如果预算不够，至少保证公共区域 (客厅/多功能室) 统一为 2.4m。
    *   **尺寸与布局 (Aesthetic Priority)**:
        *   **北侧实墙**: 2.2m 宽 (画廊主背景，墙越宽越高级)。
        *   **中间移门**: 1.4m 宽 × 2.4m 高。
        *   **南侧展示柜**: 0.5m 宽 × 2.4m 高 (通顶黑色玻璃柜)。
    *   **门体形式**: **极窄黑色边框吊轨移门 (Wall-Mounted)**。
        *   **构造**: 轨道内嵌吊顶，门扇贴着北侧墙体滑行。
        *   **搜索关键词 (Douyin/Xiaohongshu)**:
            *   **"极窄吊轨移门"**
            *   **"幽灵门"** (Ghost Door)
            *   **"单扇侧挂玻璃门"**
    *   **材质**: **单层灰玻 (Grey Glass)**。
    *   **五金**: 顶部内嵌静音轨道，底部无地轨。
    *   **五金**: 顶部预埋轨道，底部无地轨。
    *   **展示柜细节**: 黑色铝框 + 灰玻柜门 + 4000K 内嵌灯光。

### 3.5 成本与平替方案 (Cost Analysis)
*   **高性价比平替方案 (Selected Option)**: **两侧实墙 + 中间玻璃移门**
    *   **尺寸裁定**: **强烈建议调整为 2000mm (宽) × 2400mm (高)**。
        *   **理由 1 (视觉统一)**: 既然选择了砌墙，门洞就是“墙上开洞”。此时如果电竞房门(1.8m)和客厅北门(2.0m)都在一条视线上，**统一宽度会非常整齐**，治愈强迫症。
        *   **理由 2 (墙体宽裕)**: 总宽 4.2m。
            *   做 2.0m 门 -> 两侧各剩 **1.1m 实墙**。
            *   **1.1m 实墙** 依然非常宽敞，足够装开关插座、挂海报、放展示柜。完全没有“开关没地装”的顾虑。
    *   **做法**:
        *   **两侧砌轻体砖墙**: 左右各砌 1.1米 宽的墙，刷白色乳胶漆。
        *   **中间装推拉门**: 预留 2.0米 门洞，安装 **极窄黑框长虹玻璃推拉门 (两扇)**。
    *   **造价**:
        *   砌墙+油漆: 约 2000元。
        *   玻璃门 (2.0m×2.4m): 约 4.8 m² × 800元 = 3840元。
        *   **总价**: 约 **6000元** (性价比极高)。
*   **专家建议**: 如果预算充足，选全玻璃，颜值无敌；如果想省钱且实用，选“两侧砌墙+中间玻璃门”。

### 3.3 风格匹配度总结
*   **配色**: **黑色极窄边框** 方案非常完美，与全屋 **“菱花白 + 暖灰”** 基调形成高级的冷暖对比 (Black Accents)。
*   **玻璃选型 (Glass Selection)**:
    *   **客厅北门 (多功能室入口)**: 推荐 **灰玻 (Grey Glass)**。
        *   **理由**: 灰玻是透明的，能让客厅一眼看穿到画廊，空间感极强。
    *   **客厅南门 (阳台)**: 推荐 **灰玻 (Grey Glass)**。
        *   **理由**: 滤掉强光，保护隐私，同时保留景观。
    *   **电竞房门 (Zone 1)**: 推荐 **双层长虹玻璃 (Fluted Glass)**。
        *   **理由 (Necessity)**: 电竞房内部通常比较乱（线缆、外设、零食），且有时需要隐私。长虹玻璃能 **“只透光不透乱”**，就像给房间加了一层高级滤镜。如果用透明灰玻，里面的乱象会一览无余，反而破坏了画廊的美感。
    *   **专家点评**: 玻璃材质的不一致是 **功能导向** 的结果，非常合理。只要 **边框颜色 (黑色)** 和 **边框粗细 (极窄)** 保持一致，全屋风格就是统一的。
    *   **强迫症方案 (Option B)**: 如果您实在介意材质不同，可以全屋统一用 **灰玻**。前提是您必须保证电竞房内部时刻保持极其整洁，像展示厅一样。
